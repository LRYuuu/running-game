### 3.3 实现细节

#### 3.3.1 二段跳状态管理
```csharp
// 私有字段
private bool _canDoubleJump;

// 落地时重置（在 Update 中检测）
private void Update()
{
    if (IsGrounded())
    {
        _canDoubleJump = true;  // 落地重置二段跳
        _coyoteTimeTimer = _jumpConfig.coyoteTime;
    }

    // 处理跳跃输入
    if (Input.GetKeyDown(KeyCode.Space) || Input.GetKeyDown(KeyCode.K))
    {
        _jumpBufferTimer = _jumpConfig.jumpBufferTime;
        TryJump();
    }
}
```

#### 3.3.2 二段跳条件判断
```csharp
/// <summary>
/// 尝试执行跳跃（一段或二段）
/// </summary>
public void TryJump()
{
    if (_jumpBufferTimer <= 0f)
        return;

    // 一段跳条件：在地面上或有土洋时间
    if (IsGrounded() || _coyoteTimeTimer > 0f)
    {
        ExecuteJump(_jumpConfig.jumpForce);
        _jumpBufferTimer = 0f;
        _canDoubleJump = true;  // 一段跳后仍保留二段跳
        return;
    }

    // 二段跳条件：在空中且二段跳可用
    if (_canDoubleJump && !IsGrounded())
    {
        ExecuteJump(_jumpConfig.doubleJumpForce);
        _jumpBufferTimer = 0f;
        _canDoubleJump = false;  // 二段跳后不可再次跳跃
        return;
    }
}
```

## 7. 配置参数参考

### 7.1 JumpConfig 推荐值
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| jumpForce | 8-12 | 一段跳力度 |
| doubleJumpForce | 6-10 | 二段跳力度（通常略低） |

### 7.2 二段跳力度设计
```
二段跳力度 = 一段跳力度 × 0.8 ~ 0.9

理由：
- 二段跳用于修正失误，不应强于一段跳
- 避免二段跳过高导致关卡设计困难
- 保留操作技巧差异（熟练玩家会结合一段跳+二段跳）
```
