## Story

As a **玩家**,
I want **关闭游戏后最高分不丢失**,
so that **进度保存**。

---

## Acceptance Criteria

### 功能性验收标准

1. **最高分持久化**
   - Given 玩家创造了新纪录
   - When 游戏关闭后重新打开
   - Then 最高分仍然保存

2. **跨会话保存**
   - Given 玩家多次游戏
   - When 每次关闭游戏
   - Then 最高分持久化存储

3. **初始值正确**
   - Given 首次启动游戏
   - When 读取最高分
   - Then 默认值为 0

### 技术性验收标准

1. **使用 PlayerPrefs**
   - Given 最高分存储
   - When 保存到本地
   - Then 使用 Unity PlayerPrefs 系统

2. **保存时机**
   - Given 玩家死亡或打破纪录
   - When 触发保存
   - Then 立即调用 PlayerPrefs.Save()

3. **加载时机**
   - Given ScoreManager 初始化
   - When Awake 生命周期
   - Then 从 PlayerPrefs 加载最高分

4. **测试覆盖**
   - Given PlayerPrefs 持久化逻辑
   - When 编写测试
   - Then 包含保存/加载/跨会话验证

## Architecture & Design

### 现有实现

**ScoreManager.cs** 已实现：
```csharp
private const string HighScoreKey = "SquareFireline_HighScore";

private void SaveHighScore()
{
    PlayerPrefs.SetInt(HighScoreKey, HighScore);
    PlayerPrefs.Save();
}

private void LoadHighScore()
{
    HighScore = PlayerPrefs.GetInt(HighScoreKey, 0);
}
```

**保存时机**（GameState.Dying）：
```csharp
case GameState.Dying:
    if (CurrentScore > HighScore)
    {
        HighScore = CurrentScore;
        SaveHighScore();
    }
```

## Testing Requirements

### PlayMode 测试

```csharp
using NUnit.Framework;
using UnityEngine;
using UnityEngine.TestTools;
using System.Collections;
using SquareFireline.Game;

namespace SquareFireline.Tests.PlayMode.Game
{
    [TestFixture]
    public class ScoreManagerPersistenceTests
    {
        [SetUp]
        public void Setup()
        {
            // 清除之前的 PlayerPrefs 数据
            PlayerPrefs.DeleteAll();
            PlayerPrefs.Save();
            ScoreManager.ResetInstance();
        }

        [TearDown]
        public void TearDown()
        {
            PlayerPrefs.DeleteAll();
            ScoreManager.ResetInstance();
        }

        [UnityTest]
        public IEnumerator HighScore_PersistedAfterGameRestart()
        {
            // Arrange - 创建 ScoreManager 并模拟游戏
            var scoreManagerObject = new GameObject("ScoreManager");
            var scoreManager = scoreManagerObject.AddComponent<ScoreManager>();
            yield return null; // 等待一帧初始化

            // Act - 模拟游戏获得 100 分
            scoreManager.AddScore(100);

            // 模拟死亡触发保存
            GameManager.Instance.SetGameState(GameState.Dying);
            yield return null;

            // 销毁 ScoreManager 模拟游戏重启
            Object.DestroyImmediate(scoreManagerObject);
            ScoreManager.ResetInstance();

            // 创建新的 ScoreManager 模拟新游戏会话
            var newScoreManagerObject = new GameObject("ScoreManager");
            var newScoreManager = newScoreManagerObject.AddComponent<ScoreManager>();
            yield return null; // 等待 Awake 加载

            // Assert - 验证最高分被保存
            Assert.AreEqual(100, newScoreManager.HighScore);
        }

        [UnityTest]
        public IEnumerator HighScore_DefaultValueIsZero()
        {
            // Arrange
            PlayerPrefs.DeleteAll();
            PlayerPrefs.Save();

            // Act
            var scoreManagerObject = new GameObject("ScoreManager");
            var scoreManager = scoreManagerObject.AddComponent<ScoreManager>();
            yield return null;

            // Assert
            Assert.AreEqual(0, scoreManager.HighScore);
        }

        [UnityTest]
        public IEnumerator HighScore_OnlyIncreases()
        {
            // Arrange - 先保存一个最高分
            PlayerPrefs.SetInt("SquareFireline_HighScore", 50);
            PlayerPrefs.Save();

            var scoreManagerObject = new GameObject("ScoreManager");
            var scoreManager = scoreManagerObject.AddComponent<ScoreManager>();
            yield return null;

            // Act - 模拟只获得 30 分（低于最高分）
            scoreManager.AddScore(30);
            GameManager.Instance.SetGameState(GameState.Dying);
            yield return null;

            // Assert - 最高分不应下降
            Assert.AreEqual(50, scoreManager.HighScore);
        }
    }
}
```
