## Story

作为系统，
我希望管理游戏状态（开始/运行/死亡/重生），
以便控制游戏流程并确保各系统状态同步。

## Acceptance Criteria

### 功能性验收标准

1. **游戏状态定义**
   - Given 游戏启动时
   - When 系统初始化时
   - Then 游戏状态机包含 4 种状态：`Waiting`（等待开始）、`Playing`（游戏中）、`Dying`（死亡动画中）、`Respawning`（重生中）

2. **开始游戏流程**
   - Given 游戏处于 `Waiting` 状态
   - When 玩家触发开始游戏（从主菜单或重新开始）
   - Then 游戏状态切换到 `Playing`，玩家控制器激活，地图生成开始

3. **死亡触发流程**
   - Given 游戏处于 `Playing` 状态
   - When 玩家碰撞障碍物或掉落空隙
   - Then 游戏状态切换到 `Dying`，触发死亡动画/音效，禁用玩家输入

4. **重生流程**
   - Given 游戏处于 `Dying` 状态
   - When 死亡动画/延迟结束
   - Then 游戏状态切换到 `Respawning`，玩家重置到检查点位置，重置跳跃状态

5. **重生后恢复**
   - Given 游戏处于 `Respawning` 状态
   - When 重生完成
   - Then 游戏状态切换回 `Playing`，玩家控制器恢复，输入重新启用

6. **状态转换日志**
   - Given 游戏运行中
   - When 发生状态转换时
   - Then 在控制台输出状态转换日志 `[GameManager] State: Waiting → Playing`

### 技术性验收标准

1. **集中式状态机**
   - Given 游戏初始化时
   - When 创建 GameManager 单例
   - Then 使用状态机模式管理游戏状态，状态枚举集中定义

2. **事件驱动架构**
   - Given 游戏状态改变时
   - When 状态发生变化
   - Then 触发 `OnGameStateChanged` 事件，其他系统可订阅

3. **玩家死亡集成**
   - Given 玩家碰撞检测触发死亡
   - When PlayerDeathController 调用 Die()
   - Then GameManager 接收事件并切换到 `Dying` 状态

4. **检查点系统**
   - Given 玩家在游戏中移动
   - When 玩家到达安全位置
   - Then 更新当前检查点位置（用于重生）

5. **重置系统**
   - Given 玩家死亡后重生
   - When 重生流程触发
   - Then 重置玩家跳跃状态、位置、速度

6. **单例模式**
   - Given 场景加载时
   - When GameManager 实例不存在
   - Then 自动创建并设置为 DontDestroyOnLoad

#### GameManager.cs 框架
```csharp
using UnityEngine;
using System;
using SquareFireline.Player;

namespace SquareFireline.Game
{
    /// <summary>
    /// 游戏管理器 - 集中管理游戏状态机
    /// </summary>
    public class GameManager : MonoBehaviour
    {
        #region 单例
        public static GameManager Instance { get; private set; }
        #endregion

        #region 事件
        /// <summary>游戏状态改变时触发（旧状态，新状态）</summary>
        public event Action<GameState, GameState> OnGameStateChanged;
        #endregion

        #region 状态
        public GameState CurrentState { get; private set; } = GameState.Waiting;
        #endregion

        #region 依赖
        [SerializeField] private PlayerDeathController _playerDeathController;
        [SerializeField] private PlayerJumpController _playerJumpController;
        #endregion

        #region 私有字段
        private Vector3 _lastSafePosition;
        #endregion

        #region Unity 生命周期
        private void Awake()
        {
            // 单例初始化
            if (Instance != null && Instance != this)
            {
                Destroy(gameObject);
                return;
            }
            Instance = this;
            DontDestroyOnLoad(gameObject);
        }

        private void OnDestroy()
        {
            if (Instance == this)
                Instance = null;
        }

        private void OnEnable()
        {
            // 订阅玩家死亡事件
            if (_playerDeathController != null)
                _playerDeathController.OnPlayerDied += OnPlayerDied;
        }

        private void OnDisable()
        {
            if (_playerDeathController != null)
                _playerDeathController.OnPlayerDied -= OnPlayerDied;
        }
        #endregion

        #region 公共方法
        /// <summary>
        /// 开始游戏（从 Waiting → Playing）
        /// </summary>
        public void StartGame()
        {
            ChangeState(GameState.Playing);
        }

        /// <summary>
        /// 更新检查点位置
        /// </summary>
        public void UpdateCheckpoint(Vector3 position)
        {
            _lastSafePosition = position;
            Debug.Log($"[GameManager] Checkpoint updated: {_lastSafePosition}");
        }
        #endregion

        #region 私有方法
        /// <summary>
        /// 玩家死亡回调
        /// </summary>
        private void OnPlayerDied()
        {
            if (CurrentState != GameState.Playing)
                return;

            ChangeState(GameState.Dying);
            StartCoroutine(StartRespawn());
        }

        /// <summary>
        /// 改变游戏状态
        /// </summary>
        private void ChangeState(GameState newState)
        {
            if (CurrentState == newState)
                return;

            GameState oldState = CurrentState;
            CurrentState = newState;

            Debug.Log($"[GameManager] State: {oldState} → {newState}");
            OnGameStateChanged?.Invoke(oldState, newState);
        }

        /// <summary>
        /// 重生流程协程
        /// </summary>
        private System.Collections.IEnumerator StartRespawn()
        {
            // 等待死亡延迟
            yield return new WaitForSeconds(_playerDeathController.respawnDelay);

            ChangeState(GameState.Respawning);

            // 重置玩家状态
            if (_playerJumpController != null)
                _playerJumpController.ResetJumpState();

            // 重置位置
            if (_playerDeathController != null)
                _playerDeathController.RespawnToSafePosition();

            // 恢复游戏状态
            ChangeState(GameState.Playing);
        }
        #endregion
    }
}
```
