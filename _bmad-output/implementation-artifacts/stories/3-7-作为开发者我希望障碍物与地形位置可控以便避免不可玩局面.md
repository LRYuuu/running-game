## Acceptance Criteria

### 功能性验收标准

1. **空隙宽度可玩性保证**
   - Given 玩家最大跳跃距离已确定
   - When 生成空隙时
   - Then 空隙宽度不超过玩家可跳跃的最大距离 × 安全系数 (0.8)

2. **连续障碍物难度控制**
   - Given 障碍物生成配置
   - When 生成多个连续障碍物时
   - Then 连续障碍物之间的最小间隔保证玩家可以通过

3. **地形高度变化可玩性**
   - Given 地形高度变化配置
   - When 生成起伏地形时
   - Then 相邻列高度差不超过玩家可跳跃的最大高度差

4. **起始安全区域保证**
   - Given 游戏开始区域配置
   - When 生成初始地图时
   - Then 前 N 个 Chunk 保持平坦且无障碍物，让玩家熟悉操作

5. **空隙 - 障碍物组合验证**
   - Given 空隙和障碍物生成系统
   - When 生成空隙和障碍物时
   - Then 空隙两侧的障碍物位置合理，不会形成无法通过的组合

### 技术性验收标准

1. **配置验证系统**
   - Given 地图配置加载时
   - When 配置参数不合理时
   - Then 在编辑器下显示警告日志，提示开发者调整

2. **可玩性参数集中管理**
   - Given 地图配置文件
   - When 开发者调整参数时
   - Then 可以在一个地方看到所有影响可玩性的参数

3. **调试可视化**
   - Given 编辑器模式
   - When 查看地图时
   - Then 可以通过 Gizmos 看到障碍物和空隙的生成范围及关键参数

// 一段跳力度 = 7f, 二段跳力度 = 4f
// 重力 = -9.8

// 一段跳滞空时间：t = 2 * 7 / 9.8 ≈ 1.43 秒
// 一段跳水平距离：5 * 1.43 ≈ 7.15 单位

// 二段跳水平距离：5 * 0.82 ≈ 4.1 单位

### 配置验证系统实现

```csharp
// Assets/Scripts/Map/MapPlayabilityValidator.cs
using UnityEngine;

namespace RunnersJourney.Map
{
    /// <summary>
    /// 地图可玩性验证器 - 验证地图配置参数的可玩性
    /// </summary>
    public static class MapPlayabilityValidator
    {
        /// <summary>
        /// 验证地图配置参数的可玩性
        /// </summary>
        public static void ValidateConfig(TilemapMapConfig config)
        {
            if (!config.enablePlayabilityValidation) return;

            bool hasWarning = false;

            // 1. 验证空隙宽度
            if (config.maxGapWidth > config.maxGapWidthPlayable)
            {
                Debug.LogWarning($"[MapPlayabilityValidator] 警告：最大空隙宽度 ({config.maxGapWidth}) 超过可玩值 ({config.maxGapWidthPlayable})，玩家可能无法跳过！");
                hasWarning = true;
            }

            // 2. 验证障碍物间隔
            if (config.minObstacleGap < config.minPlayableObstacleGap)
            {
                Debug.LogWarning($"[MapPlayabilityValidator] 警告：最小障碍物间隔 ({config.minObstacleGap}) 小于可玩值 ({config.minPlayableObstacleGap})，玩家可能无法通过！");
                hasWarning = true;
            }

            // 3. 验证高度差
            if (config.heightVariation > config.maxHeightDifference * 2)
            {
                Debug.LogWarning($"[MapPlayabilityValidator] 警告：高度变化幅度过大 ({config.heightVariation})，可能导致地形不可玩！");
                hasWarning = true;
            }

            // 4. 验证起始区域
            if (config.flatChunkCount < config.safeStartChunkCount)
            {
                Debug.LogWarning($"[MapPlayabilityValidator] 警告：起始平坦区域不足 ({config.flatChunkCount} < {config.safeStartChunkCount})，玩家可能没有足够时间熟悉操作！");
                hasWarning = true;
            }

            // 5. 验证空隙 - 障碍物间隔
            if (config.minGapInterval * config.chunkWidth < config.minObstacleGap + config.maxGapWidth)
            {
                Debug.LogWarning($"[MapPlayabilityValidator] 警告：空隙间隔可能不足以形成安全的障碍物布局！");
                hasWarning = true;
            }

            if (!hasWarning)
            {
                Debug.Log("[MapPlayabilityValidator] 配置验证通过，所有参数符合可玩性要求！");
            }
        }
    }
}
```

#if UNITY_EDITOR
private void OnDrawGizmos()
{
    if (config == null) return;

    // ... 现有的 Chunk 边界和地面高度线 ...

    // 新增：显示可玩性相关调试信息

    // 1. 障碍物生成范围（蓝色）
    Gizmos.color = new Color(0, 0, 1, 0.3f);
    for (int x = minGeneratedChunk * config.chunkWidth; x < maxGeneratedChunk * config.chunkWidth; x++)
    {
        if (GetColumnHeight(x) > 0)
        {
            // 检查该位置是否可以生成障碍物
            bool canSpawn = ShouldSpawnObstacleDebug(x);
            if (canSpawn)
            {
                Gizmos.DrawCube(new Vector3(x, config.baseHeight + 1.5f, 0), Vector3.one * 0.5f);
            }
        }
    }

    // 2. 危险区域标记（红色半透明框）
    Gizmos.color = new Color(1, 0, 0, 0.2f);
    int prevHeight = GetColumnHeight(minGeneratedChunk * config.chunkWidth);
    for (int x = minGeneratedChunk * config.chunkWidth + 1; x < maxGeneratedChunk * config.chunkWidth; x++)
    {
        int currentHeight = GetColumnHeight(x);

        // 检查高度差是否过大
        if (Mathf.Abs(currentHeight - prevHeight) > config.maxHeightDifference)
        {
            Gizmos.DrawCube(new Vector3(x, currentHeight / 2f, 0), new Vector3(1, currentHeight, 1));
        }

        // 检查空隙宽度是否可玩
        if (IsInGap(x))
        {
            int gapWidth = gapEndWorldX - gapStartWorldX;
            if (gapWidth > config.maxGapWidthPlayable)
            {
                Gizmos.DrawCube(new Vector3(x, config.baseHeight / 2f, 0), new Vector3(1, config.baseHeight, 1));
            }
        }

        prevHeight = currentHeight;
    }

    // 3. 显示配置参数 HUD
    Handles.Label(transform.position + Vector3.up * (config.baseHeight + 3),
        $"Gap: {config.minGapWidth}-{config.maxGapWidth} (max playable: {config.maxGapWidthPlayable})");
    Handles.Label(transform.position + Vector3.up * (config.baseHeight + 4),
        $"Obs Gap: {config.minObstacleGap}-{config.maxObstacleGap} (min playable: {config.minPlayableObstacleGap})");
}
#endif

## 配置说明

### 可玩性参数推荐值

基于玩家跳跃能力的测试结果：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| `maxGapWidthPlayable` | 8 | 玩家可跳跃的最大空隙宽度（安全系数 0.8） |
| `minPlayableObstacleGap` | 3 | 障碍物之间最小可玩间隔 |
| `maxHeightDifference` | 1 | 相邻列最大高度差（保证可以跳上） |
| `safeStartChunkCount` | 3 | 起始安全区域 Chunk 数量 |
