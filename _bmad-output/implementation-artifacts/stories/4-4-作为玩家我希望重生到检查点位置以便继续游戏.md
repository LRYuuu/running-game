# Story 4-4: 作为玩家，我希望重生到起始位置，以便重新开始游戏

**Epic:** Epic 4 - 游戏流程管理
**优先级:** P0
**估算:** 2h
**状态:** done

---

## Story

作为玩家，
我希望死亡后重生到游戏起始位置，
以便重新开始游戏而不是继续到检查点。

---

## Acceptance Criteria

### 功能性验收标准

1. **死亡后重生到起始位置**
   - Given 玩家在游戏中死亡
   - When 重生流程完成
   - Then 玩家回到游戏开始时的起始位置

2. **地图完全重置**
   - Given 玩家死亡后重生
   - When 重生流程完成
   - Then 地图被完全清理并重新生成

3. **障碍物完全重置**
   - Given 玩家死亡后重生
   - When 重生流程完成
   - Then 所有障碍物被清理并重新生成

4. **分数重置**
   - Given 玩家死亡
   - When 重生流程完成
   - Then 当前分数被重置为 0

5. **初始位置保存**
   - Given 游戏开始（StartGame 调用）
   - When 玩家按下跳跃键或开始移动
   - Then 当前玩家位置被保存为起始位置

### 技术性验收标准

1. **PlayerDeathController 起始位置管理**
   - Given 游戏开始时
   - When GameManager.StartGame() 被调用
   - Then PlayerDeathController.ResetStartPosition() 保存当前位置

2. **地图重置流程**
   - Given 玩家死亡
   - When GameManager.StartRespawn() 协程执行
   - Then TilemapEndlessMapGenerator.Cleanup() 被调用

3. **地图重新生成**
   - Given 地图被清理
   - When 重生流程继续
   - Then TilemapEndlessMapGenerator.Initialize() 被调用

4. **与 ScoreManager 集成**
   - Given 玩家死亡
   - When 游戏状态变为 Waiting
   - Then ScoreManager.ResetScore() 自动被调用

### 重生流程时序图

```
PlayerDeathController → GameManager → TilemapEndlessMapGenerator → ScoreManager

1. 玩家碰撞障碍物
   ↓
2. Die() → OnPlayerDied.Invoke()
   ↓
3. OnPlayerDied() → ChangeState(Dying) → StartCoroutine(StartRespawn())
   ↓
4. 等待 respawnDelay 秒
   ↓
5. ChangeState(Respawning)
   ↓
6. _mapGenerator.Cleanup()  // 清理地图和障碍物
   ↓
7. _playerJumpController.ResetJumpState()
   ↓
8. _playerDeathController.Respawn()  // 重置位置到 _startPosition
   ↓
9. _mapGenerator.Initialize()  // 重新生成地图
   ↓
10. ChangeState(Playing)
    ↓
11. (自动) ScoreManager.OnGameStateChanged() → ResetScore()
```

## Implementation Notes

### 代码修改摘要

**GameManager.cs 修改：**

```csharp
// 添加 using
using SquareFireline.Map;

// 添加依赖引用
[Header("依赖引用")]
[SerializeField] private TilemapEndlessMapGenerator _mapGenerator;

// 修改 StartGame()
public void StartGame()
{
    // 重置玩家起始位置（游戏开始时保存当前位置）
    if (_playerDeathController != null)
    {
        _playerDeathController.ResetStartPosition();
    }
    ChangeState(GameState.Playing);
}

// 修改 StartRespawn()
private IEnumerator StartRespawn()
{
    // 等待死亡延迟
    float respawnDelay = _playerDeathController != null
        ? _playerDeathController.RespawnDelay : 1f;
    yield return new WaitForSeconds(respawnDelay);

    ChangeState(GameState.Respawning);

    // 1. 清理地图和障碍物
    if (_mapGenerator != null)
    {
        _mapGenerator.Cleanup();
    }

    // 2. 重置玩家跳跃状态
    if (_playerJumpController != null)
    {
        _playerJumpController.ResetJumpState();
    }

    // 3. 重置玩家位置到起始点
    if (_playerDeathController != null)
    {
        _playerDeathController.Respawn();
    }

    // 4. 重新生成地图
    if (_mapGenerator != null)
    {
        _mapGenerator.Initialize();
    }

    // 5. 恢复游戏状态
    ChangeState(GameState.Playing);
}
```

## Testing Requirements

### EditMode 测试

创建 `Assets/Tests/EditMode/Game/GameResetTests.cs`

```csharp
using NUnit.Framework;
using UnityEngine;
using SquareFireline.Game;
using SquareFireline.Player;

namespace SquareFireline.Tests.EditMode.Game
{
    [TestFixture]
    public class GameResetTests
    {
        private GameObject _gameManagerObject;
        private GameManager _gameManager;
        private GameObject _playerObject;
        private PlayerDeathController _playerDeathController;

        [SetUp]
        public void SetUp()
        {
            _gameManagerObject = new GameObject("GameManager");
            _gameManager = _gameManagerObject.AddComponent<GameManager>();

            _playerObject = new GameObject("Player");
            _playerDeathController = _playerObject.AddComponent<PlayerDeathController>();
        }

        [TearDown]
        public void TearDown()
        {
            GameObject.DestroyImmediate(_gameManagerObject);
            GameObject.DestroyImmediate(_playerObject);
            GameManager.ResetInstance();
        }

        [Test]
        public void StartGame_ShouldSavePlayerStartPosition()
        {
            // Arrange
            _playerObject.transform.position = new Vector3(5f, 10f, 0f);

            // Act
            _gameManager.StartGame();

            // Assert
            Assert.AreEqual(new Vector3(5f, 10f, 0f), _playerDeathController.GetStartPosition());
        }

        [Test]
        public void Respawn_ShouldReturnToStartPosition()
        {
            // Arrange
            Vector3 startPosition = new Vector3(5f, 10f, 0f);
            _playerObject.transform.position = startPosition;
            _gameManager.StartGame();

            // Move player to different position
            _playerObject.transform.position = new Vector3(20f, 15f, 0f);

            // Act
            _playerDeathController.Respawn();

            // Assert
            Assert.AreEqual(startPosition, _playerObject.transform.position);
        }
    }
}
```
