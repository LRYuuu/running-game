## Story

As a **玩家**,

so that **有成就感**。

## Acceptance Criteria

### 功能性验收标准

1. **破纪录提示显示**
   - Given 当前分数超过历史最高分
   - When CheckHighScore() 返回 true
   - Then 显示"新纪录！"或"New Record!"提示

2. **提示位置**
   - Given 游戏内 UI 显示
   - When 破纪录提示出现
   - Then 提示位于屏幕中央或分数附近（醒目位置）

3. **提示样式**
   - Given 破纪录提示
   - When 显示在屏幕上
   - Then 使用醒目的颜色和动画效果（如金色放大字体）

4. **提示持续时间**
   - Given 破纪录提示显示
   - When 2-3 秒后
   - Then 提示自动淡出或消失

5. **游戏结束显示（可选扩展）**
   - Given 玩家死亡且打破纪录
   - When 游戏结束界面显示
   - Then 显示"新纪录！"标志

### 技术性验收标准

1. **事件驱动**
   - Given ScoreManager.CheckHighScore() 被调用
   - When 返回 true（新纪录）
   - Then 触发破纪录事件通知 UI 显示提示

2. **UI Toolkit 实现**
   - Given 破纪录提示 UI
   - When 渲染
   - Then 使用 UI Toolkit (UXML/USS) 实现

3. **动画效果**
   - Given 破纪录提示
   - When 显示时
   - Then 使用淡入/放大动画增强视觉效果

4. **性能优化**
   - Given 破纪录不常发生
   - When 提示显示
   - Then 使用协程或 USS 动画自动清理，避免内存泄漏

## Implementation Notes

### 代码修改

#### ScoreManager.cs 添加事件

```csharp
#region 事件
/// <summary>分数改变时触发（当前分数，最高分）</summary>
public event Action<int, int> OnScoreChanged;
/// <summary>打破新纪录时触发（新最高分）</summary>
public event Action<int> OnNewRecord;
#endregion

/// <summary>
/// 检查是否打破纪录
/// </summary>
public bool CheckHighScore()
{
    if (CurrentScore > HighScore)
    {
        HighScore = CurrentScore;
        SaveHighScore();
        OnScoreChanged?.Invoke(CurrentScore, HighScore);

        // 触发破纪录事件
        OnNewRecord?.Invoke(HighScore);

        return true;
    }
    return false;
}
```

#### InGameUI.cs 添加提示逻辑

```csharp
#region 私有字段
private Label _newRecordLabel;
private Coroutine _hideNewRecordCoroutine;
#endregion

#region Unity 生命周期
private void OnEnable()
{
    // ... 现有代码

    // 订阅破纪录事件
    if (ScoreManager.Instance != null)
    {
        ScoreManager.Instance.OnNewRecord += OnNewRecord;
    }
}

private void OnDisable()
{
    // ... 现有代码

    if (ScoreManager.Instance != null)
    {
        ScoreManager.Instance.OnNewRecord -= OnNewRecord;
    }
}
#endregion

#region 公共方法
/// <summary>
/// 显示破纪录提示
/// </summary>
public void ShowNewRecordHint()
{
    if (_newRecordContainer != null)
    {
        _newRecordContainer.style.display = DisplayStyle.Flex;
        Debug.Log("[InGameUI] New record hint shown");

        // 取消之前的隐藏协程
        if (_hideNewRecordCoroutine != null)
        {
            StopCoroutine(_hideNewRecordCoroutine);
        }

        // 2.5 秒后自动隐藏
        _hideNewRecordCoroutine = StartCoroutine(HideNewRecordAfterDelay(2.5f));
    }
}
#endregion

#region 私有方法
/// <summary>
/// 延迟隐藏破纪录提示
/// </summary>
private System.Collections.IEnumerator HideNewRecordAfterDelay(float delay)
{
    yield return new WaitForSeconds(delay);

    if (_newRecordContainer != null)
    {
        _newRecordContainer.style.display = DisplayStyle.None;
        Debug.Log("[InGameUI] New record hint hidden");
    }

    _hideNewRecordCoroutine = null;
}

/// <summary>
/// 破纪录事件回调
/// </summary>
private void OnNewRecord(int newHighScore)
{
    Debug.Log($"[InGameUI] New record: {newHighScore}");
    ShowNewRecordHint();
}
#endregion
```

## Testing Requirements

### EditMode 测试

```csharp
using NUnit.Framework;
using UnityEngine;
using SquareFireline.Game;

namespace SquareFireline.Tests.EditMode.Game
{
    [TestFixture]
    public class ScoreManagerNewRecordTests
    {
        private ScoreManager _scoreManager;
        private GameObject _scoreManagerObject;

        [SetUp]
        public void Setup()
        {
            _scoreManagerObject = new GameObject("ScoreManager");
            _scoreManager = _scoreManagerObject.AddComponent<ScoreManager>();
        }

        [TearDown]
        public void TearDown()
        {
            Object.DestroyImmediate(_scoreManagerObject);
            ScoreManager.ResetInstance();
        }

        [Test]
        public void CheckHighScore_NewRecord_TriggerEvent()
        {
            // Arrange
            bool eventTriggered = false;
            int receivedScore = 0;
            _scoreManager.OnNewRecord += (score) => {
                eventTriggered = true;
                receivedScore = score;
            };

            // Act
            _scoreManager.AddScore(100);
            bool isNewRecord = _scoreManager.CheckHighScore();

            // Assert
            Assert.IsTrue(isNewRecord);
            Assert.IsTrue(eventTriggered);
            Assert.AreEqual(100, receivedScore);
        }

        [Test]
        public void CheckHighScore_NotNewRecord_NoEventTriggered()
        {
            // Arrange
            _scoreManager.HighScore = 100;
            bool eventTriggered = false;
            _scoreManager.OnNewRecord += (score) => {
                eventTriggered = true;
            };

            // Act
            _scoreManager.AddScore(50);
            bool isNewRecord = _scoreManager.CheckHighScore();

            // Assert
            Assert.IsFalse(isNewRecord);
            Assert.IsFalse(eventTriggered);
        }
    }
}
```
