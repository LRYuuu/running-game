# Story 5-1: 作为玩家，我可以看到主界面并开始游戏，以便进入体验

**Epic:** Epic 5 - UI 系统
**优先级:** P0
**估算:** 2h
**状态:** ready-for-dev

---

## Story

As a **玩家**,
I want **看到主界面并开始游戏**,
so that **进入游戏体验**。

---

## Acceptance Criteria

### 功能性验收标准

1. **主界面显示**
   - Given 游戏启动完成
   - When 场景加载完成
   - Then 显示主界面 UI，包含游戏标题和"开始游戏"按钮

2. **开始游戏功能**
   - Given 玩家在主界面
   - When 玩家点击"开始游戏"按钮
   - Then 游戏状态切换到 Playing，主界面隐藏，游戏开始

3. **UI 布局**
   - Given 主界面显示
   - When UI 渲染完成
   - Then 游戏标题位于屏幕上方中央，"开始游戏"按钮位于屏幕中央偏下位置

4. **按钮交互反馈**
   - Given 主界面显示
   - When 鼠标悬停/触摸按钮
   - Then 按钮有视觉反馈（颜色变化/放大效果）

5. **点击音效**
   - Given 音频系统已实现（Epic 7）
   - When 玩家点击按钮
   - Then 播放 UI 点击音效（未来扩展）

### 技术性验收标准

1. **UI Toolkit 实现**
   - Given 主界面创建
   - When UI 系统初始化
   - Then 使用 UI Toolkit (UXML/USS) 实现，而非 legacy Canvas

2. **数据绑定模式**
   - Given 游戏状态改变
   - When GameState 从 Waiting 变为 Playing
   - Then UI 自动响应状态变化（订阅 OnGameStateChanged 事件）

3. **单例 UI 管理器**
   - Given 场景切换
   - When 新场景加载
   - Then UIManager 保持持久化（DontDestroyOnLoad）或通过事件重建

4. **配置驱动**
   - Given UI 参数需要调整
   - When 平衡 UI 效果
   - Then 通过 UIConfig ScriptableObject 配置（未来扩展）

5. **响应式设计**
   - Given 不同分辨率
   - When 屏幕尺寸改变
   - Then UI 正确缩放（使用 Canvas Scaler 或 UI Toolkit 缩放）

#### UIManager.cs 框架
```csharp
using UnityEngine;
using System;
using SquareFireline.Game;

namespace SquareFireline.UI
{
    /// <summary>
    /// UI 管理器 - 集中管理所有 UI 界面
    /// </summary>
    public class UIManager : MonoBehaviour
    {
        #region 单例
        public static UIManager Instance { get; private set; }
        #endregion

        #region 事件
        /// <summary>UI 状态改变时触发</summary>
        public event Action<UIState, UIState> OnUIStateChanged;
        #endregion

        #region 属性
        public UIState CurrentState { get; private set; } = UIState.MainMenu;
        #endregion

        #region 序列化字段
        [Header("UI 引用")]
        [Tooltip("主界面 UIDocument")]
        [SerializeField] private UIDocument mainMenuUI;

        [Header("游戏管理器引用")]
        [Tooltip("游戏管理器（用于获取状态）")]
        [SerializeField] private GameManager gameManager;
        #endregion

        #region 私有字段
        private MainMenuUI _mainMenuController;
        #endregion

        #region Unity 生命周期
        private void Awake()
        {
            // 单例初始化
            if (Instance != null && Instance != this)
            {
                Destroy(gameObject);
                return;
            }
            Instance = this;
            DontDestroyOnLoad(gameObject);

            Initialize();
        }

        private void OnDestroy()
        {
            if (Instance == this)
                Instance = null;

            // 取消订阅
            if (gameManager != null)
                gameManager.OnGameStateChanged -= OnGameStateChanged;
        }

        private void Start()
        {
            // 初始显示主菜单
            ShowMainMenu();
        }
        #endregion

        #region 公共方法
        /// <summary>
        /// 显示主菜单
        /// </summary>
        public void ShowMainMenu()
        {
            if (_mainMenuController != null)
                _mainMenuController.Show();

            ChangeState(UIState.MainMenu);
        }

        /// <summary>
        /// 隐藏主菜单
        /// </summary>
        public void HideMainMenu()
        {
            if (_mainMenuController != null)
                _mainMenuController.Hide();
        }

        /// <summary>
        /// 显示游戏内 UI
        /// </summary>
        public void ShowInGameUI()
        {
            // 后续 Story 5-2 实现
            ChangeState(UIState.InGame);
        }
        #endregion

        #region 私有方法
        /// <summary>
        /// 初始化 UI 系统
        /// </summary>
        private void Initialize()
        {
            // 查找主界面控制器
            if (mainMenuUI != null)
            {
                _mainMenuController = mainMenuUI.GetComponent<MainMenuUI>();
                if (_mainMenuController == null)
                {
                    _mainMenuController = mainMenuUI.gameObject.AddComponent<MainMenuUI>();
                }
            }

            // 订阅游戏状态事件
            if (gameManager != null)
            {
                gameManager.OnGameStateChanged += OnGameStateChanged;
            }
        }

        /// <summary>
        /// 游戏状态改变回调
        /// </summary>
        private void OnGameStateChanged(GameState oldState, GameState newState)
        {
            switch (newState)
            {
                case GameState.Waiting:
                    ShowMainMenu();
                    break;
                case GameState.Playing:
                    HideMainMenu();
                    ShowInGameUI();
                    break;
                case GameState.Dying:
                case GameState.Respawning:
                    // 不处理，保持游戏内 UI 显示或根据需求调整
                    break;
            }
        }

        /// <summary>
        /// 改变 UI 状态
        /// </summary>
        private void ChangeState(UIState newState)
        {
            if (CurrentState == newState)
                return;

            UIState oldState = CurrentState;
            CurrentState = newState;

            Debug.Log($"[UIManager] UI State: {oldState} → {newState}");
            OnUIStateChanged?.Invoke(oldState, newState);
        }
        #endregion
    }
}
```

#### MainMenuUI.cs 框架
```csharp
using UnityEngine;
using UnityEngine.UIElements;
using SquareFireline.Game;

namespace SquareFireline.UI
{
    /// <summary>
    /// 主界面 UI 控制器
    /// </summary>
    public class MainMenuUI : MonoBehaviour
    {
        #region 序列化字段
        [Header("UI 文档引用")]
        [Tooltip("主界面 UXML 文档")]
        [SerializeField] private UIDocument uiDocument;
        #endregion

        #region 私有字段
        private VisualElement _root;
        private Button _startButton;
        private Label _titleLabel;
        private GameManager _gameManager;
        #endregion

        #region Unity 生命周期
        private void Awake()
        {
            // 获取 GameManager 引用
            _gameManager = GameManager.Instance;

            // 获取 UI 根元素
            if (uiDocument != null)
            {
                _root = uiDocument.rootVisualElement;
                InitializeUIElements();
                SetupEventListeners();
            }
        }
        #endregion

        #region 公共方法
        /// <summary>
        /// 显示主界面
        /// </summary>
        public void Show()
        {
            if (_root != null)
                _root.style.display = DisplayStyle.Flex;
        }

        /// <summary>
        /// 隐藏主界面
        /// </summary>
        public void Hide()
        {
            if (_root != null)
                _root.style.display = DisplayStyle.None;
        }
        #endregion

        #region 私有方法
        /// <summary>
        /// 初始化 UI 元素引用
        /// </summary>
        private void InitializeUIElements()
        {
            // 获取按钮引用
            _startButton = _root.Q<Button>("start-button");

            // 获取标题引用
            _titleLabel = _root.Q<Label>("title-label");
        }

        /// <summary>
        /// 设置事件监听
        /// </summary>
        private void SetupEventListeners()
        {
            // 绑定开始游戏按钮点击事件
            if (_startButton != null)
            {
                _startButton.clicked += HandleStartClick;

                // 悬停效果
                _startButton.RegisterCallback<PointerEnterEvent>(evt =>
                {
                    _startButton.AddToClassList("hover");
                });

                _startButton.RegisterCallback<PointerLeaveEvent>(evt =>
                {
                    _startButton.RemoveFromClassList("hover");
                });
            }
        }

        /// <summary>
        /// 处理开始游戏按钮点击
        /// </summary>
        private void HandleStartClick()
        {
            Debug.Log("[MainMenuUI] Start button clicked");

            // 通知 GameManager 开始游戏
            if (_gameManager != null)
            {
                _gameManager.StartGame();
            }
        }
        #endregion
    }
}
```

            // Act
            var instance = UIManager.Instance;

            // Assert
            Assert.AreEqual(_uiManager, instance);
        }

        {

            _uiManager.OnUIStateChanged += (old, @new) => {
                if (@new == UIState.MainMenu) mainMenuShown = true;

            _uiManager.ShowMainMenu();

            Assert.IsTrue(mainMenuShown);

            _uiManager.ShowMainMenu();

            // Act

            // Assert

// 世界暂停状态
private bool _isWorldPaused = false;

/// <summary>
/// 设置世界暂停状态（暂停/恢复地图和背景滚动）
/// </summary>
public void SetWorldPaused(bool paused)
{
    if (_isWorldPaused == paused)
        return;

    _isWorldPaused = paused;

    // 暂停/恢复地图滚动
    if (_mapGenerator != null)
    {
        _mapGenerator.SetScrollPaused(paused);
    }

    // 暂停/恢复背景滚动
    var backgrounds = FindObjectsOfType<MonoBehaviour>();
    foreach (var bg in backgrounds)
    {
        if (bg is ParallaxBackground parallaxBg)
        {
            parallaxBg.SetScrollPaused(paused);
        }
        else if (bg is ParallaxBackgroundTilemap tilemapBg)
        {
            tilemapBg.SetScrollPaused(paused);
        }
    }
}

// 在 ChangeState 中根据 GameState 自动调用
private void ChangeState(GameState newState)
{
    // ...
    if (newState == GameState.Waiting)
    {
        SetWorldPaused(true);  // Waiting 状态：暂停世界滚动
    }
    else if (newState == GameState.Playing)
    {
        SetWorldPaused(false); // Playing 状态：恢复世界滚动
    }
    // ...
}

### 技术决策

**方案 A：独立场景（推荐）**
- 创建 `MainMenu.unity` 作为启动场景（索引 0）
- 场景 0：MainMenu → 点击开始后加载场景 1（游戏场景）
- 优点：结构清晰，资源隔离
- 缺点：场景加载时间

**方案 B：预制体加载**
- 在启动场景加载 `MainMenuUI.prefab`
- 游戏开始时销毁
- 优点：简单快速
- 缺点：资源管理复杂

**推荐方案 A** - 符合 Unity 标准场景流程
