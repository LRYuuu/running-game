## Dev Agent Guardrails

### 技术需求

1. **问题根因分析**

   当前 `PlayerDeathController.cs` 只检测障碍物死亡：
   - `OnCollisionEnter2D` - 通过 Layer/Tag 检测障碍物
   - `OnTriggerEnter2D` - 通过 Layer/Tag 检测障碍物

   **关键发现：**
   - 空隙 Tile (`gapTopTile`, `gapCenterTile`) 原本在 **Ground Tilemap** 上
   - Ground Tilemap: Tag="Untagged", Layer=Default
   - `IsObstacle()` 检测的是 Layer 或 Tag "Obstacle"
   - 所以玩家踩到空隙 Tile 时检测不到死亡

2. **解决方案**

   **核心思路：让空隙 Tile 使用与障碍物相同的检测机制**

   - 将空隙 Tile 从 `groundTilemap` 移到 `obstacleTilemap`
   - `obstacleTilemap` 有 "Obstacle" Layer/Tag
   - 现有 `IsObstacle()` 检测自动生效，无需修改 PlayerDeathController

3. **需要修改的文件**

   | 文件 | 修改内容 |
   |------|----------|
   | `TilemapEndlessMapGenerator.cs` | `GenerateGapColumn()` 改为在 `obstacleTilemap` 上放置空隙 Tile |
   | `TilemapEndlessMapGenerator.cs` | `ClearColumn()` 正确清理 `obstacleTilemap` 上的空隙 Tile |

## Implementation Guide

### Step 1: 修改 GenerateGapColumn 方法

将空隙 Tile 从 `groundTilemap` 改为 `obstacleTilemap`：

```csharp
private void GenerateGapColumn(int worldX, int groundHeight)
{
    // 空隙 Tile 放到障碍物 Tilemap 上，以便触发死亡检测（与障碍物相同的方式）
    // 顶部水流 Tile（地面高度位置）
    if (groundHeight > 0)
    {
        Vector3Int gapTopPos = new Vector3Int(worldX, groundHeight - 1, 0);
        obstacleTilemap.SetTile(gapTopPos, config.gapTopTile);
    }

    // 中间水流 Tile（从 y=0 到 y=groundHeight-2）
    for (int y = 0; y < groundHeight - 1; y++)
    {
        Vector3Int gapCenterPos = new Vector3Int(worldX, y, 0);
        obstacleTilemap.SetTile(gapCenterPos, config.gapCenterTile);
    }
}
```

### Step 2: 修改 ClearColumn 方法

确保正确清理 `obstacleTilemap` 上的空隙 Tile：

```csharp
private void ClearColumn(int x)
{
    // ...
    // 清除障碍物 Tilemap 上的所有 Tile（包括障碍物和空隙）
    if (obstacleTilemap != null)
    {
        // 清除障碍物（在草坪层上方）
        Vector3Int obsPos = new Vector3Int(x, columnHeight, 0);
        obstacleTilemap.SetTile(obsPos, null);

        // 清除空隙 Tile（可能占据从 y=0 到 y=columnHeight-1）
        for (int y = 0; y < columnHeight; y++)
        {
            Vector3Int gapPos = new Vector3Int(x, y, 0);
            obstacleTilemap.SetTile(gapPos, null);
        }
    }
    // ...
}
```
