# Story 5-3: 作为玩家，我可以看到历史最高分，以便有挑战目标

**Epic:** Epic 5 - UI 系统
**优先级:** P1
**估算:** 1h
**状态:** ready-for-dev

---

## Story

As a **玩家**,
I want **在游戏中看到历史最高分**,
so that **有挑战目标并追踪自己的进步**。

---

## Acceptance Criteria

### 功能性验收标准

1. **最高分显示**
   - Given 游戏正在进行中（GameState = Playing）
   - When 游戏内 UI 显示
   - Then 屏幕右上角显示历史最高分

2. **最高分更新**
   - Given 当前分数超过历史最高分
   - When CheckHighScore() 被调用
   - Then UI 立即更新显示新的最高分

3. **新纪录视觉反馈（可选扩展）**
   - Given 玩家打破了历史最高分
   - When 游戏结束（Dying 状态）
   - Then 显示"新纪录！"提示（Story 5-3 可选扩展）

4. **最高分持久化**
   - Given 玩家关闭游戏
   - When 再次启动游戏
   - Then 历史最高分仍然保留（通过 PlayerPrefs 保存）

5. **UI 位置**
   - Given 游戏进行中
   - When 最高分显示在屏幕上
   - Then 最高分位于屏幕右上角（与当前分数对称）

### 技术性验收标准

1. **事件驱动更新**
   - Given ScoreManager.HighScore 变化
   - When OnScoreChanged 事件触发
   - Then InGameUI 同时更新当前分数和最高分显示

2. **UI Toolkit 一致性**
   - Given 最高分 UI 创建
   - When UI 渲染
   - Then 使用 UI Toolkit (UXML/USS)，与 Story 5-2 保持一致

3. **空值保护**
   - Given highscore-label 未找到
   - When 游戏运行
   - Then 不抛出异常，仅记录 Warning 日志

4. **性能优化**
   - Given 最高分不常变化
   - When 分数更新
   - Then 仅在最高分实际变化时更新 UI（缓存比较）

5. **可配置显示**
   - Given 某些模式不需要显示最高分
   - When InGameUI 初始化
   - Then 通过 `showHighScore` 字段控制是否显示

## Implementation Notes

### 代码修改

#### InGameUI.cs 修改点

```csharp
// 在 InGameUI.cs 中添加缓存字段
private int _lastDisplayedHighScore = -1; // 避免重复更新最高分

// 修改 InitializeUIElements 方法
private void InitializeUIElements()
{
    if (_root == null)
    {
        Debug.LogWarning("[InGameUI] InitializeUIElements called but _root is null");
        return;
    }

    // 获取分数标签引用
    _scoreLabel = _root.Q<Label>("score-label");

    if (_scoreLabel == null)
    {
        Debug.LogWarning("[InGameUI] score-label not found in UXML");
    }
    else
    {
        Debug.Log($"[InGameUI] score-label found, initial text: {_scoreLabel.text}");
    }

    // 获取最高分标签引用
    if (showHighScore)
    {
        _highScoreLabel = _root.Q<Label>("highscore-label");
        if (_highScoreLabel == null)
        {
            Debug.LogWarning("[InGameUI] highscore-label not found in UXML");
        }
        else
        {
            Debug.Log($"[InGameUI] highscore-label found, initial text: {_highScoreLabel.text}");
        }
    }
}

// 修改 UpdateScoreDisplay 方法
public void UpdateScoreDisplay(int currentScore, int highScore)
{
    // 更新当前分数（已有逻辑）
    if (currentScore != _lastDisplayedScore && _scoreLabel != null)
    {
        _lastDisplayedScore = currentScore;
        _scoreLabel.text = $"分数：{currentScore}";
        Debug.Log($"[InGameUI] Score updated: {_scoreLabel.text}");
    }

    // 更新最高分（新增逻辑）
    if (showHighScore && highScore != _lastDisplayedHighScore && _highScoreLabel != null)
    {
        _lastDisplayedHighScore = highScore;
        _highScoreLabel.text = $"最高分：{highScore}";
        Debug.Log($"[InGameUI] HighScore updated: {_highScoreLabel.text}");
    }
}
```

```csharp
using NUnit.Framework;
using UnityEngine;
using SquareFireline.UI;
using SquareFireline.Game;

namespace SquareFireline.Tests.EditMode.UI
{
    [TestFixture]
    public class HighScoreTests
    {
        private ScoreManager _scoreManager;
        private GameObject _scoreManagerObject;

        [SetUp]
        public void Setup()
        {
            // 创建 ScoreManager 测试实例
            _scoreManagerObject = new GameObject("ScoreManager");
            _scoreManager = _scoreManagerObject.AddComponent<ScoreManager>();
        }

        [TearDown]
        public void TearDown()
        {
            Object.DestroyImmediate(_scoreManagerObject);
            ScoreManager.ResetInstance();
        }

        [Test]
        public void CheckHighScore_NewHighScore_ReturnsTrue()
        {
            // Arrange
            _scoreManager.AddScore(100);

            // Act
            bool isNewRecord = _scoreManager.CheckHighScore();

            // Assert
            Assert.IsTrue(isNewRecord);
            Assert.AreEqual(100, _scoreManager.HighScore);
        }

        [Test]
        public void CheckHighScore_NoNewHighScore_ReturnsFalse()
        {
            // Arrange
            _scoreManager.HighScore = 100;
            _scoreManager.AddScore(50);

            // Act
            bool isNewRecord = _scoreManager.CheckHighScore();

            // Assert
            Assert.IsFalse(isNewRecord);
            Assert.AreEqual(100, _scoreManager.HighScore);
        }

        [Test]
        public void CheckHighScore_UpdateScoreDisplay_BothValuesUpdated()
        {
            // Arrange
            bool scoreUpdated = false;
            bool highScoreUpdated = false;
            _scoreManager.OnScoreChanged += (score, highScore) => {
                scoreUpdated = true;
                highScoreUpdated = (highScore > 0);
            };

            // Act
            _scoreManager.AddScore(100);
            _scoreManager.CheckHighScore();

            // Assert
            Assert.IsTrue(scoreUpdated);
            Assert.IsTrue(highScoreUpdated);
        }
    }
}
```

```csharp
using NUnit.Framework;
using UnityEngine;
using UnityEngine.TestTools;
using System.Collections;

namespace SquareFireline.Tests.PlayMode.UI
{
    public class HighScorePersistenceTests
    {
        [UnityTest]
        public IEnumerator HighScore_PersistedAcrossSessions()
        {
            // Arrange
            string highScoreKey = "SquareFireline_HighScore";
            int testHighScore = 500;
            PlayerPrefs.SetInt(highScoreKey, testHighScore);
            PlayerPrefs.Save();

            // Act - 创建新的 ScoreManager
            var scoreManagerObject = new GameObject("ScoreManager");
            var scoreManager = scoreManagerObject.AddComponent<ScoreManager>();
            yield return null; // 等待 Awake 执行

            // Assert
            Assert.AreEqual(testHighScore, scoreManager.HighScore);

            // Cleanup
            PlayerPrefs.DeleteKey(highScoreKey);
            Object.Destroy(scoreManagerObject);
        }

        [UnityTest]
        public IEnumerator CheckHighScore_SavesNewRecord()
        {
            // Arrange
            string highScoreKey = "SquareFireline_HighScore";
            PlayerPrefs.DeleteKey(highScoreKey);

            var scoreManagerObject = new GameObject("ScoreManager");
            var scoreManager = scoreManagerObject.AddComponent<ScoreManager>();
            yield return null;

            // Act - 添加分数并检查
            scoreManager.AddScore(300);
            scoreManager.CheckHighScore();
            yield return null;

            // Assert
            Assert.AreEqual(300, scoreManager.HighScore);
            Assert.AreEqual(300, PlayerPrefs.GetInt(highScoreKey));

            // Cleanup
            PlayerPrefs.DeleteKey(highScoreKey);
            Object.Destroy(scoreManagerObject);
        }
    }
}
```
