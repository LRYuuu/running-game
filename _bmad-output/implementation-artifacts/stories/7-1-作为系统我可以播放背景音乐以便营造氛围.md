## Acceptance Criteria

### 功能验收标准

1. **背景音乐播放**
   - Given 游戏开始运行
   - When 进入游戏场景时
   - Then 自动播放背景音乐

2. **音乐循环播放**
   - Given 背景音乐正在播放
   - When 音乐播放完毕时
   - Then 自动循环继续播放

3. **音量控制**
   - Given 游戏进行中
   - When 调整音量设置时
   - Then 背景音乐音量随之改变

4. **场景切换保持**
   - Given 背景音乐正在播放
   - When 场景切换时（如主菜单→游戏）
   - Then 背景音乐持续播放不中断

5. **音频资源管理**
   - Given 有多个背景音乐曲目
   - When 游戏运行时
   - Then 可以根据场景/状态播放对应曲目

### 技术验收标准

1. **AudioSource 组件使用**
   - Given Unity 音频系统
   - When 播放背景音乐时
   - Then 使用 AudioSource 组件配合 AudioClip

2. **单例管理模式**
   - Given 背景音乐需要在场景间持续
   - When 创建音频管理器时
   - Then 使用单例模式 + DontDestroyOnLoad

3. **配置驱动**
   - Given 需要调整音量参数
   - When 使用 ScriptableObject 配置时
   - Then 可以配置默认音量和最大/最小值

4. **代码规范**
   - Given 项目代码规范
   - When 创建新脚本时
   - Then 遵循命名空间、字段命名、注释规范

5. **测试覆盖**
   - Given EditMode 测试框架
   - When 完成实现后
   - Then 为核心逻辑编写单元测试

using UnityEngine;

namespace RunnersJourney.Audio
{
    [CreateAssetMenu(fileName = "AudioConfig", menuName = "Runner's Journey/Audio Config")]
    public class AudioConfig : ScriptableObject
    {
        [Header("音量设置")]
        [Tooltip("背景音乐默认音量 (0-1)")]
        [Range(0f, 1f)]
        public float bgmVolume = 0.8f;

        [Tooltip("音效默认音量 (0-1)")]
        [Range(0f, 1f)]
        public float sfxVolume = 0.6f;

        [Tooltip("总音量控制 (0-1)")]
        [Range(0f, 1f)]
        public float masterVolume = 1.0f;
    }
}

```csharp
// AudioManager.cs - 音频管理器单例
using UnityEngine;
using UnityEngine.Events;

namespace RunnersJourney.Audio
{
    /// <summary>
    /// 音频管理器单例
    /// 负责管理背景音乐和音效的播放
    /// </summary>
    public class AudioManager : MonoBehaviour
    {
        private static AudioManager _instance;

        public static AudioManager Instance
        {
            get
            {
                if (_instance == null)
                {
                    _instance = FindObjectOfType<AudioManager>();
                    if (_instance == null)
                    {
                        var go = new GameObject("AudioManager");
                        _instance = go.AddComponent<AudioManager>();
                    }
                }
                return _instance;
            }
        }

        [Header("音频配置")]
        [Tooltip("音频配置文件")]
        [SerializeField] private AudioConfig audioConfig;

        [Header("音频源")]
        [Tooltip("背景音乐音频源")]
        [SerializeField] private AudioSource bgmSource;

        [Tooltip("音效音频源")]
        [SerializeField] private AudioSource sfxSource;

        // 音频剪辑缓存
        private AudioClip _currentBGM;

        // 事件
        public UnityEvent<float> OnBGMVolumeChanged = new();

        private void Awake()
        {
            if (_instance != null && _instance != this)
            {
                Destroy(gameObject);
                return;
            }
            _instance = this;
            DontDestroyOnLoad(gameObject);

            InitializeAudioSources();
        }

        private void Start()
        {
            if (audioConfig != null)
            {
                SetBGMVolume(audioConfig.bgmVolume);
                SetSFXVolume(audioConfig.sfxVolume);
            }
        }

        /// <summary>
        /// 初始化音频源
        /// </summary>
        private void InitializeAudioSources()
        {
            // 创建或获取 AudioSource 组件
            if (bgmSource == null)
            {
                bgmSource = gameObject.AddComponent<AudioSource>();
            }
            if (sfxSource == null)
            {
                sfxSource = gameObject.AddComponent<AudioSource>();
            }

            // 配置 BGM 音频源
            bgmSource.loop = true;
            bgmSource.playOnAwake = false;

            // 配置 SFX 音频源
            sfxSource.loop = false;
            sfxSource.playOnAwake = false;
        }

        /// <summary>
        /// 播放背景音乐
        /// </summary>
        /// <param name="clip">要播放的音乐剪辑</param>
        public void PlayBGM(AudioClip clip)
        {
            if (clip == null)
            {
                Debug.LogWarning("[AudioManager] 尝试播放 null 背景音乐");
                return;
            }

            _currentBGM = clip;
            bgmSource.clip = clip;
            bgmSource.Play();
            Debug.Log($"[AudioManager] 开始播放 BGM: {clip.name}");
        }

        /// <summary>
        /// 停止背景音乐
        /// </summary>
        public void StopBGM()
        {
            if (bgmSource.isPlaying)
            {
                bgmSource.Stop();
                _currentBGM = null;
                Debug.Log("[AudioManager] 停止播放 BGM");
            }
        }

        /// <summary>
        /// 设置背景音乐音量
        /// </summary>
        /// <param name="volume">音量值 (0-1)</param>
        public void SetBGMVolume(float volume)
        {
            volume = Mathf.Clamp01(volume);
            bgmSource.volume = volume;
            OnBGMVolumeChanged?.Invoke(volume);
        }

        /// <summary>
        /// 设置音效音量
        /// </summary>
        /// <param name="volume">音量值 (0-1)</param>
        public void SetSFXVolume(float volume)
        {
            volume = Mathf.Clamp01(volume);
            sfxSource.volume = volume;
        }

        /// <summary>
        /// 播放音效
        /// </summary>
        /// <param name="clip">要播放的音效剪辑</param>
        public void PlaySFX(AudioClip clip)
        {
            if (clip == null)
            {
                Debug.LogWarning("[AudioManager] 尝试播放 null 音效");
                return;
            }
            sfxSource.PlayOneShot(clip);
        }

        /// <summary>
        /// 获取当前 BGM 音量
        /// </summary>
        public float GetCurrentBGMVolume() => bgmSource.volume;
    }
}
```

## Implementation Notes

### 实现建议

1. **音频资源准备**
   - 准备 1-2 首背景音乐（BGM）
   - 格式建议：.wav 或 .mp3
   - 时长：1-3 分钟循环曲目
   - 放置在 `Assets/Audio/Music/` 目录

2. **AudioSource 配置**
   - `loop = true`: 背景音乐循环播放
   - `playOnAwake = false`: 手动控制播放时机
   - `spatialBlend = 0`: 2D 音频（无空间衰减）

3. **场景切换保持**
   - 使用 `DontDestroyOnLoad(gameObject)`
   - AudioManager 单例确保全局唯一

4. **音量持久化（可选）**
   - 使用 PlayerPrefs 保存音量设置
   - 启动游戏时读取上次设置

### 与现有系统集成

1. **GameManager**
   - 在 StartGame() 时调用 AudioManager.PlayBGM()
   - 可以从 GameConfig 引用 BGM 资源

2. **UIManager**
   - 可在设置 UI 中添加音量滑块
   - 滑块变化时调用 AudioManager.SetBGMVolume()

3. **AudioConfig**
   - 使用 ScriptableObject 存储音量配置
   - 可在 Inspector 中调整默认值

### 音频资源建议

| 场景 | 推荐风格 | 时长 |
|------|---------|------|
| 主菜单 | 轻松、舒缓 | 1-2 分钟 |
| 游戏进行 | 动感、节奏感 | 2-3 分钟 |
| 死亡/失败 | 低沉、短暂 | 10-30 秒 |
| 胜利/破纪录 | 欢快、激昂 | 15-30 秒 |

### 测试要点

1. **EditMode 测试**
   - AudioManager 单例初始化正确
   - 音量设置在有效范围内
   - 事件触发正确

2. **PlayMode 测试**
   - 场景切换时音乐持续播放
   - 音量调整即时生效
   - 循环播放正常工作
