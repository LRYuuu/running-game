## Acceptance Criteria

### 功能验收标准

1. **跳跃时播放音效**
   - Given 游戏进行中
   - When 玩家执行跳跃（包括一段跳和二段跳）
   - Then 播放跳跃音效

2. **音效音量控制**
   - Given 跳跃音效正在播放
   - When 调整音效音量设置时
   - Then 跳跃音效音量随之改变

3. **快速响应**
   - Given 玩家按下跳跃键
   - Then 音效响应延迟 <100ms

4. **同时跳跃处理**
   - Given 多个跳跃动作连续发生
   - When 前一个音效尚未播放完毕
   - Then 使用 PlayOneShot 确保每个音效都能完整播放

### 技术验收标准

1. **使用现有 AudioManager**
   - Given AudioManager 单例已存在
   - When 播放跳跃音效时
   - Then 调用 AudioManager.PlaySFX() 方法

2. **音效资源管理**
   - Given 需要播放跳跃音效
   - When 调用 PlaySFX 时
   - Then 使用 AudioClip 资源

3. **与 PlayerJumpController 集成**
   - Given 玩家跳跃控制器
   - When ExecuteJump 方法执行时
   - Then 触发跳跃音效播放

4. **代码规范**
   - Given 项目代码规范
   - When 修改现有脚本时
   - Then 遵循命名空间、字段命名、注释规范

### 方案 B：创建独立的 JumpSFX 组件

**创建 `JumpSFX.cs`：**

```csharp
using UnityEngine;
using RunnersJourney.Audio;

namespace RunnersJourney.Audio
{
    /// <summary>
    /// 跳跃音效控制器
    /// 监听跳跃事件并播放对应音效
    /// </summary>
    public class JumpSFX : MonoBehaviour
    {
        #region 序列化字段
        [Header("音频剪辑")]
        [Tooltip("一段跳音效")]
        [SerializeField] private AudioClip _jumpSFX;

        [Tooltip("二段跳音效（可选，与一段跳区分）")]
        [SerializeField] private AudioClip _doubleJumpSFX;

        [Header("音量设置")]
        [Tooltip("音效音量 (0-1)")]
        [Range(0f, 1f)]
        [SerializeField] private float volume = 0.6f;
        #endregion

        #region 私有字段
        private PlayerJumpController _playerJumpController;
        #endregion

        #region Unity 生命周期
        private void Awake()
        {
            _playerJumpController = GetComponent<PlayerJumpController>();
            if (_playerJumpController == null)
            {
                Debug.LogWarning("[JumpSFX] 未找到 PlayerJumpController 组件");
            }
        }

        private void OnEnable()
        {
            // 订阅跳跃事件（如果有）
            if (_playerJumpController != null)
            {
                // 可以通过事件或轮询方式检测跳跃
            }
        }
        #endregion

        #region 公共方法
        /// <summary>
        /// 播放跳跃音效
        /// </summary>
        /// <param name="isDoubleJump">是否为二段跳</param>
        public void PlayJumpSound(bool isDoubleJump = false)
        {
            AudioClip clipToPlay = isDoubleJump && _doubleJumpSFX != null
                ? _doubleJumpSFX
                : _jumpSFX;

            if (clipToPlay == null)
            {
                Debug.LogWarning("[JumpSFX] 尝试播放 null 音效");
                return;
            }

            if (AudioManager.Instance != null)
            {
                AudioManager.Instance.PlaySFX(clipToPlay, volume);
            }
            else
            {
                Debug.LogWarning("[JumpSFX] AudioManager 实例不存在");
            }
        }
        #endregion
    }
}
```
