# Story 5-2: 作为玩家，我可以在游戏中看到当前分数，以便了解进度

**Epic:** Epic 5 - UI 系统
**优先级:** P0
**估算:** 1h
**状态:** ready-for-dev

---

## Story

As a **玩家**,
I want **在游戏中看到当前分数**,
so that **了解游戏进度**。

---

## Acceptance Criteria

### 功能性验收标准

1. **分数显示**
   - Given 游戏正在进行中（GameState = Playing）
   - When 分数发生变化
   - Then UI 实时显示当前分数

2. **分数更新**
   - Given 分数增加
   - When ScoreManager 触发 OnScoreChanged 事件
   - Then UI 立即更新显示的分数值

3. **分数格式化**
   - Given 当前分数为 1000
   - When 显示在 UI 上
   - Then 显示为 "1000" 或 "1,000"（带千位分隔符，可选）

4. **UI 位置**
   - Given 游戏进行中
   - When 分数显示在屏幕上
   - Then 分数位于屏幕左上角或右上角（不遮挡游戏区域）

5. **游戏结束时重置**
   - Given 玩家死亡
   - When 游戏状态变为 Dying
   - Then 分数显示重置为 0（下一局开始）

### 技术性验收标准

1. **事件订阅模式**
   - Given ScoreManager 分数变化
   - When OnScoreChanged 事件触发
   - Then 游戏内 UI 订阅并响应事件更新显示

2. **UI Toolkit 实现**
   - Given 游戏内 UI 创建
   - When UI 渲染
   - Then 使用 UI Toolkit (UXML/USS) 实现，与 Story 5-1 保持一致

3. **空值保护**
   - Given ScoreManager 或 UI 引用未设置
   - When 游戏运行
   - Then 使用默认值，不抛出异常，记录 Warning 日志

4. **性能优化**
   - Given 分数频繁更新
   - When 每帧更新 UI
   - Then 避免不必要的 UI 刷新（仅分数值变化时更新）

## Implementation Notes

### 代码模板

#### InGameUI.cs 框架

```csharp
using UnityEngine;
using UnityEngine.UIElements;
using SquareFireline.Game;

namespace SquareFireline.UI
{
    /// <summary>
    /// 游戏内 UI 控制器 - 显示当前分数
    /// </summary>
    public class InGameUI : MonoBehaviour
    {
        #region 序列化字段
        [Header("UI 文档引用")]
        [Tooltip("游戏内 UI UXML 文档")]
        [SerializeField] private UIDocument uiDocument;

        [Header("可选：最高分显示（Story 5-3 扩展）")]
        [Tooltip("是否显示最高分")]
        [SerializeField] private bool showHighScore = false;
        #endregion

        #region 私有字段
        private VisualElement _root;
        private Label _scoreLabel;
        private Label _highScoreLabel;
        private int _lastDisplayedScore = -1; // 用于避免重复更新
        #endregion

        #region Unity 生命周期
        private void Awake()
        {
            // 获取 UI 根元素
            if (uiDocument != null)
            {
                _root = uiDocument.rootVisualElement;
                InitializeUIElements();
            }
            else
            {
                Debug.LogWarning("[InGameUI] UIDocument not assigned");
            }
        }

        private void OnEnable()
        {
            // 订阅分数变化事件
            if (ScoreManager.Instance != null)
            {
                ScoreManager.Instance.OnScoreChanged += OnScoreChanged;
                // 初始化显示当前分数
                UpdateScoreDisplay(
                    ScoreManager.Instance.CurrentScore,
                    ScoreManager.Instance.HighScore
                );
            }
        }

        private void OnDisable()
        {
            // 取消订阅
            if (ScoreManager.Instance != null)
            {
                ScoreManager.Instance.OnScoreChanged -= OnScoreChanged;
            }
        }
        #endregion

        #region 公共方法
        /// <summary>
        /// 显示游戏内 UI
        /// </summary>
        public void Show()
        {
            if (_root != null)
                _root.style.display = DisplayStyle.Flex;
        }

        /// <summary>
        /// 隐藏游戏内 UI
        /// </summary>
        public void Hide()
        {
            if (_root != null)
                _root.style.display = DisplayStyle.None;
        }

        /// <summary>
        /// 更新分数显示
        /// </summary>
        /// <param name="currentScore">当前分数</param>
        /// <param name="highScore">最高分</param>
        public void UpdateScoreDisplay(int currentScore, int highScore)
        {
            // 避免重复更新（性能优化）
            if (currentScore == _lastDisplayedScore)
                return;

            _lastDisplayedScore = currentScore;

            if (_scoreLabel != null)
            {
                // 可选：使用千位分隔符格式化
                _scoreLabel.text = $"分数：{currentScore}";
                // 或者：_scoreLabel.text = $"分数：{currentScore:N0}";
            }

            if (_highScoreLabel != null && showHighScore)
            {
                _highScoreLabel.text = $"最高分：{highScore}";
            }
        }
        #endregion

        #region 私有方法
        /// <summary>
        /// 初始化 UI 元素引用
        /// </summary>
        private void InitializeUIElements()
        {
            if (_root == null)
                return;

            // 获取分数标签引用
            _scoreLabel = _root.Q<Label>("score-label");

            if (_scoreLabel == null)
            {
                Debug.LogWarning("[InGameUI] score-label not found in UXML");
            }

            // 获取最高分标签引用（可选）
            if (showHighScore)
            {
                _highScoreLabel = _root.Q<Label>("highscore-label");
            }
        }

        /// <summary>
        /// 分数变化回调
        /// </summary>
        private void OnScoreChanged(int currentScore, int highScore)
        {
            UpdateScoreDisplay(currentScore, highScore);
        }
        #endregion
    }
}
```

### UIManager 集成

在 `UIManager.cs` 中添加游戏内 UI 管理：

```csharp
// 在 UIManager 类中添加以下字段和方法

#region 私有字段
private InGameUI _inGameUI;
#endregion

#region Unity 生命周期
private void Start()
{
    // 查找游戏内 UI 控制器
    var inGameUIObj = GameObject.Find("InGameUIDocument");
    if (inGameUIObj != null)
    {
        _inGameUI = inGameUIObj.GetComponent<InGameUI>();
        if (_inGameUI == null)
        {
            _inGameUI = inGameUIObj.AddComponent<InGameUI>();
        }
    }
}
#endregion

#region 公共方法
/// <summary>
/// 显示游戏内 UI（完善 Story 5-1 中的预留方法）
/// </summary>
public override void ShowInGameUI()
{
    if (CurrentUIState == UIState.InGame)
        return;

    var oldState = CurrentUIState;
    CurrentUIState = UIState.InGame;

    // 显示游戏内 UI
    if (_inGameUI != null)
    {
        _inGameUI.Show();
    }

    Debug.Log("[UIManager] In-game UI shown");
    OnUIStateChanged?.Invoke(oldState, CurrentUIState);
}
#endregion

#region 私有方法
private void OnGameStateChanged(GameState oldState, GameState newState)
{
    Debug.Log($"[UIManager] Game state changed: {oldState} → {newState}");

    switch (newState)
    {
        case GameState.Waiting:
            // 等待开始状态 → 显示主界面，隐藏游戏内 UI
            ShowMainMenu();
            if (_inGameUI != null) _inGameUI.Hide();
            break;

        case GameState.Playing:
            // 游戏中状态 → 隐藏主界面，显示游戏内 UI
            HideMainMenu();
            ShowInGameUI();
            break;

        case GameState.Dying:
        case GameState.Respawning:
            // 死亡/重生状态 → 保持游戏内 UI 显示或根据需求调整
            break;
    }
}
#endregion
```

### Key Technical Decisions

1. **延迟初始化 UI 元素**
   - 问题：`InGameUI.OnEnable` 执行时 `UIDocument.rootVisualElement` 可能还未创建
   - 解决：使用 `InitializeUIAfterDelay()` 协程等待一帧后初始化

2. **ScoreManager 初始化时序问题**
   - 问题：`InGameUI.OnEnable` 执行时 `ScoreManager.Instance` 可能为 null
   - 解决：在 `Update()` 中轮询检查，直到 `ScoreManager.Instance` 可用时自动订阅

3. **场景配置修复**
   - 直接修改 `login.unity` 场景文件配置 UIDocument.sourceAsset
   - 使用正确的 GUID 引用：`{fileID: 9197627878781567013, guid: b8db01f3222cfa047b041dfd2954240b}`

### Lessons Learned

1. **UIDocument 初始化时序** - `rootVisualElement` 在 `OnEnable` 时可能未就绪，需要延迟一帧获取
