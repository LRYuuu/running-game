# Story 6.2: 作为玩家，我希望通过游戏内页面中能够选择不同的群系，以便有更多选择

**Epic:** Epic 6 - 地图生成扩展
**优先级:** P0
**估时:** 6h
**状态:** done

---

## Story

作为玩家，
我希望通过游戏内页面中能够选择不同的群系模式，
以便有更多选择。

**四种群系模式:**
1. **草地** - 固定使用草坪群系
2. **沙漠** - 固定使用沙漠群系
3. **雪地** - 固定使用雪地群系
4. **混合群系** - 随距离自动切换（草地→沙漠→雪地）

---

## Acceptance Criteria

### 功能验收标准

1. **群系选择 UI 界面**
   - Given 玩家在主界面
   - When 点击"选择群系"按钮
   - Then 显示群系选择面板，展示所有可用的群系选项

2. **四种群系模式可选**
   - Given 群系选择面板已打开
   - When 查看群系选项
   - Then 可以看到 4 个选项：
     - 草地（固定草坪群系）
     - 沙漠（固定沙漠群系）
     - 雪地（固定雪地群系）
     - 混合群系（随距离自动切换：草地→沙漠→雪地）

3. **群系预览功能**
   - Given 群系选择面板已打开
   - When 悬停或点击某个群系选项
   - Then 显示该群系的视觉预览（颜色主题/特色元素说明）
   - And 混合模式显示"随距离自动切换"的说明

4. **确认选择后保存**
   - Given 玩家选择了一个群系模式
   - When 点击"确认"按钮
   - Then 保存选择并开始游戏，游戏使用选定模式生成地图

5. **默认群系回退**
   - Given 玩家首次游玩或未选择群系
   - When 开始游戏
   - Then 使用默认的草地群系

6. **混合模式正确触发**
   - Given 玩家选择了混合群系模式
   - When 游戏进行中
   - Then BiomeManager 使用 BiomeSequence 配置随距离自动切换群系

### 技术验收标准

1. **群系选择数据持久化**
   - Given 玩家选择了群系模式
   - When 使用 PlayerPrefs 保存
   - Then 下次打开游戏时记住上次选择

2. **UI 遵循现有规范**
   - Given 群系选择 UI
   - When 使用 UI Toolkit 实现
   - Then 遵循 1920x1080 参考分辨率，使用 StandardPanelSettings

3. **与 BiomeManager 集成**
   - Given 玩家确认选择
   - When 调用 BiomeManager API
   - Then 正确切换到选定模式（固定群系或混合序列）

4. **两种模式支持**
   - Given 玩家选择固定群系
   - When 调用 BiomeManager.SetBiome()
   - Then 使用固定单一群系
   - Given 玩家选择混合群系
   - When 调用 BiomeManager.SetBiomeSequence()
   - Then 使用序列配置随距离自动切换

5. **移动端适配**
   - Given 移动端屏幕较小
   - When UI 使用 ScaleWithScreenSize
   - Then 在不同分辨率下正确显示

```csharp
// BiomeSelectionData.cs - 群系选择数据持久化
using UnityEngine;

namespace RunnersJourney.Game
{
    /// <summary>
    /// 群系选择数据持久化
    /// 使用 PlayerPrefs 保存玩家选择的群系模式
    /// </summary>
    public static class BiomeSelectionData
    {
        private const string SelectedBiomeKey = "SelectedBiome";
        private const string HasSelectedKey = "HasSelectedBiome";

        // 群系模式键值
        public const string ModeGrassland = "grassland";      // 固定草地
        public const string ModeDesert = "desert";            // 固定沙漠
        public const string ModeSnowland = "snowland";        // 固定雪地
        public const string ModeSequence = "sequence";        // 混合群系

        /// <summary>
        /// 保存玩家选择的群系模式
        /// </summary>
        public static void SaveSelection(string modeKey)
        {
            PlayerPrefs.SetString(SelectedBiomeKey, modeKey);
            PlayerPrefs.SetInt(HasSelectedKey, 1);
            PlayerPrefs.Save();
            Debug.Log($"[BiomeSelectionData] 保存选择：{modeKey}");
        }

        /// <summary>
        /// 加载玩家上次选择的群系模式
        /// </summary>
        public static string LoadSelection()
        {
            if (PlayerPrefs.HasKey(SelectedBiomeKey))
            {
                return PlayerPrefs.GetString(SelectedBiomeKey);
            }
            return null;
        }

        /// <summary>
        /// 是否已经选择过群系
        /// </summary>
        public static bool HasSelectedBefore()
        {
            return PlayerPrefs.GetInt(HasSelectedKey, 0) == 1;
        }

        /// <summary>
        /// 重置群系选择
        /// </summary>
        public static void ResetSelection()
        {
            PlayerPrefs.DeleteKey(SelectedBiomeKey);
            PlayerPrefs.DeleteKey(HasSelectedKey);
            Debug.Log("[BiomeSelectionData] 重置群系选择");
        }

        /// <summary>
        /// 是否为混合模式
        /// </summary>
        public static bool IsSequenceMode(string modeKey)
        {
            return modeKey == ModeSequence;
        }
    }
}
```

```csharp
// BiomeSelectionPanel.cs - 群系选择 UI 控制器
using UnityEngine;
using UnityEngine.UIElements;
using System.Collections.Generic;

namespace RunnersJourney.UI
{
    /// <summary>
    /// 群系选择面板控制器
    /// 支持四种模式：草地、沙漠、雪地、混合群系
    /// </summary>
    public class BiomeSelectionPanel : MonoBehaviour
    {
        [Header("UI 组件")]
        [Tooltip("UIDocument 组件")]
        [SerializeField] private UIDocument uiDocument;

        [Header("群系配置")]
        [Tooltip("固定模式可用的群系配置列表")]
        [SerializeField] private List<BiomeConfig> availableBiomes = new List<BiomeConfig>();

        [Tooltip("混合模式使用的群系序列配置")]
        [SerializeField] private BiomeSequence biomeSequence;

        [Header("序列化字段")]
        [Tooltip("群系选项按钮的命名前缀")]
        [SerializeField] private string biomeButtonPrefix = "biome-button-";

        [Tooltip("确认按钮名称")]
        [SerializeField] private string confirmButtonName = "confirm-button";

        [Tooltip("取消按钮名称")]
        [SerializeField] private string cancelButtonName = "cancel-button";

        [Tooltip("预览图片名称")]
        [SerializeField] private string previewImageName = "preview-image";

        [Tooltip("描述文本标签名称")]
        [SerializeField] private string descriptionLabelName = "description-label";

        // 私有字段
        private VisualElement _root;
        private string _selectedModeKey;
        private Dictionary<string, Button> _modeButtons = new Dictionary<string, Button>();
        private Image _previewImage;
        private Label _descriptionLabel;

        // 当前选择的群系配置引用
        private BiomeConfig _selectedBiomeConfig;

        // 模式描述
        private static readonly Dictionary<string, string> ModeDescriptions = new Dictionary<string, string>
        {
            { BiomeSelectionData.ModeGrassland, "翠绿的草原，适合新手练习" },
            { BiomeSelectionData.ModeDesert, "金色的沙漠，中等难度挑战" },
            { BiomeSelectionData.ModeSnowland, "冰雪覆盖，高难度考验" },
            { BiomeSelectionData.ModeSequence, "随距离自动切换：草原→沙漠→雪地" }
        };

        private void OnEnable()
        {
            if (uiDocument == null)
            {
                uiDocument = GetComponent<UIDocument>();
            }

            if (uiDocument != null && uiDocument.rootVisualElement != null)
            {
                _root = uiDocument.rootVisualElement;
                InitializeUI();
            }
        }

        /// <summary>
        /// 初始化 UI 组件
        /// </summary>
        private void InitializeUI()
        {
            // 获取预览组件
            _previewImage = _root.Q<Image>(previewImageName);
            _descriptionLabel = _root.Q<Label>(descriptionLabelName);

            // 注册固定模式群系按钮
            foreach (var biome in availableBiomes)
            {
                if (biome == null) continue;

                string modeKey = GetModeKeyForBiome(biome.biomeName);
                string buttonName = $"{biomeButtonPrefix}{modeKey}";
                var button = _root.Q<Button>(buttonName);

                if (button != null)
                {
                    _modeButtons[modeKey] = button;
                    BiomeConfig biomeRef = biome;
                    button.clicked += () => OnModeSelected(modeKey, biomeRef);
                    Debug.Log($"[BiomeSelectionPanel] 注册群系按钮：{buttonName}");
                }
                else
                {
                    Debug.LogWarning($"[BiomeSelectionPanel] 未找到群系按钮：{buttonName}");
                }
            }

            // 注册混合模式按钮
            string seqButtonName = $"{biomeButtonPrefix}{BiomeSelectionData.ModeSequence}";
            var seqButton = _root.Q<Button>(seqButtonName);
            if (seqButton != null)
            {
                _modeButtons[BiomeSelectionData.ModeSequence] = seqButton;
                seqButton.clicked += () => OnModeSelected(BiomeSelectionData.ModeSequence, null);
                Debug.Log($"[BiomeSelectionPanel] 注册混合模式按钮：{seqButtonName}");
            }

            // 注册确认/取消按钮
            var confirmButton = _root.Q<Button>(confirmButtonName);
            if (confirmButton != null)
            {
                confirmButton.clicked += OnConfirmClicked;
            }

            var cancelButton = _root.Q<Button>(cancelButtonName);
            if (cancelButton != null)
            {
                cancelButton.clicked += OnCancelClicked;
            }

            // 加载上次选择
            LoadLastSelection();
        }

        /// <summary>
        /// 根据群系名称获取模式键值
        /// </summary>
        private string GetModeKeyForBiome(string biomeName)
        {
            switch (biomeName.ToLower())
            {
                case "grassland": return BiomeSelectionData.ModeGrassland;
                case "desert": return BiomeSelectionData.ModeDesert;
                case "snowland": return BiomeSelectionData.ModeSnowland;
                default: return biomeName.ToLower();
            }
        }

        /// <summary>
        /// 加载上次选择的模式
        /// </summary>
        private void LoadLastSelection()
        {
            string lastKey = BiomeSelectionData.LoadSelection();
            if (!string.IsNullOrEmpty(lastKey) && _modeButtons.ContainsKey(lastKey))
            {
                _selectedModeKey = lastKey;
                UpdateButtonVisual();
                UpdatePreview();
            }
            else
            {
                // 默认选择草地
                _selectedModeKey = BiomeSelectionData.ModeGrassland;
                UpdateButtonVisual();
                UpdatePreview();
            }
        }

        /// <summary>
        /// 玩家选择群系模式
        /// </summary>
        private void OnModeSelected(string modeKey, BiomeConfig biome)
        {
            _selectedModeKey = modeKey;
            _selectedBiomeConfig = biome;
            UpdateButtonVisual();
            UpdatePreview();
        }

        /// <summary>
        /// 更新按钮选中状态
        /// </summary>
        private void UpdateButtonVisual()
        {
            foreach (var kvp in _modeButtons)
            {
                if (kvp.Key == _selectedModeKey)
                {
                    kvp.Value.EnableInClassList("selected", true);
                }
                else
                {
                    kvp.Value.EnableInClassList("selected", false);
                }
            }
        }

        /// <summary>
        /// 更新预览显示
        /// </summary>
        private void UpdatePreview()
        {
            if (_selectedBiomeConfig != null)
            {
                // 固定模式：使用群系的背景色
                if (_previewImage != null)
                {
                    _previewImage.style.backgroundColor = _selectedBiomeConfig.backgroundColor;
                }
            }
            else if (_selectedModeKey == BiomeSelectionData.ModeSequence)
            {
                // 混合模式：使用渐变色或默认色
                if (_previewImage != null)
                {
                    _previewImage.style.backgroundColor = new Color(0.5f, 0.5f, 0.8f);
                }
            }

            // 更新描述
            if (_descriptionLabel != null && !string.IsNullOrEmpty(_selectedModeKey))
            {
                if (ModeDescriptions.ContainsKey(_selectedModeKey))
                {
                    _descriptionLabel.text = ModeDescriptions[_selectedModeKey];
                }
            }
        }

        /// <summary>
        /// 确认选择
        /// </summary>
        private void OnConfirmClicked()
        {
            if (!string.IsNullOrEmpty(_selectedModeKey))
            {
                // 保存选择
                BiomeSelectionData.SaveSelection(_selectedModeKey);
                Debug.Log($"[BiomeSelectionPanel] 确认选择：{_selectedModeKey}");

                // 关闭面板
                Hide();
            }
        }

        /// <summary>
        /// 取消选择
        /// </summary>
        private void OnCancelClicked()
        {
            // 恢复上次选择
            LoadLastSelection();
            Hide();
        }

        /// <summary>
        /// 显示面板
        /// </summary>
        public void Show()
        {
            gameObject.SetActive(true);
            LoadLastSelection();
        }

        /// <summary>
        /// 隐藏面板
        /// </summary>
        public void Hide()
        {
            gameObject.SetActive(false);
        }
    }
}
```

```csharp
        [Header("生物群系设置")]
        [Tooltip("是否允许玩家自选群系模式")]
        public bool allowBiomeSelection = true;

        private void StartGameWithBiome()
        {
            // 检查玩家是否选择了群系模式
            if (allowBiomeSelection && BiomeSelectionData.HasSelectedBefore())
            {
                string selectedMode = BiomeSelectionData.LoadSelection();
                if (!string.IsNullOrEmpty(selectedMode))
                {
                    // 判断是否为混合模式
                    if (BiomeSelectionData.IsSequenceMode(selectedMode))
                    {
                        // 混合模式：使用群系序列
                        var sequence = GetBiomeSequence();
                        if (sequence != null && BiomeManager.Instance != null)
                        {
                            BiomeManager.Instance.SetBiomeSequence(sequence);
                            Debug.Log("[GameManager] 使用混合群系模式");
                        }
                    }
                    else
                    {
                        // 固定模式：使用选定群系
                        var biomeConfig = FindBiomeConfigByMode(selectedMode);
                        if (biomeConfig != null && BiomeManager.Instance != null)
                        {
                            BiomeManager.Instance.SetBiome(biomeConfig);
                            Debug.Log($"[GameManager] 使用固定群系：{selectedMode}");
                        }
                    }
                    return;
                }
            }

            // 如果玩家未选择，使用 BiomeManager 的默认配置
            Debug.Log("[GameManager] 使用默认群系配置");
        }

        private BiomeSequence GetBiomeSequence()
        {
            var sequences = Resources.LoadAll<BiomeSequence>("Biomes/Sequences");
            if (sequences.Length > 0)
            {
                return sequences[0];
            }
            return null;
        }

        private BiomeConfig FindBiomeConfigByMode(string modeKey)
        {
            var allBiomes = Resources.LoadAll<BiomeConfig>("Biomes");
            foreach (var biome in allBiomes)
            {
                string biomeMode = GetModeKeyForBiome(biome.biomeName);
                if (biomeMode == modeKey)
                {
                    return biome;
                }
            }
            return null;
        }

        private string GetModeKeyForBiome(string biomeName)
        {
            switch (biomeName.ToLower())
            {
                case "grassland": return BiomeSelectionData.ModeGrassland;
                case "desert": return BiomeSelectionData.ModeDesert;
                case "snowland": return BiomeSelectionData.ModeSnowland;
                default: return biomeName.ToLower();
            }
        }
    }
}
```
