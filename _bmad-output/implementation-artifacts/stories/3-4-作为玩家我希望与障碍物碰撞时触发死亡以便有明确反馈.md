## 实现摘要

### 1. 玩家死亡控制器 (`PlayerDeathController.cs`)

**文件:** `Assets/Scripts/Player/PlayerDeathController.cs`

**核心功能:**

```csharp
// 障碍物检测（LayerMask 优先，Tag  fallback）
[SerializeField] private LayerMask _obstacleLayer = 0;

private bool IsObstacle(GameObject obj)
{
    // 方法 1: 通过 Layer 检测
    if (_obstacleLayer.value > 0)
    {
        return ((_obstacleLayer.value & (1 << obj.layer)) != 0);
    }
    // 方法 2: 通过 Tag 检测（后备方案）
    return obj.CompareTag("Obstacle");
}

// 碰撞检测
private void OnCollisionEnter2D(Collision2D collision)
{
    if (IsObstacle(collision.gameObject))
    {
        HandleObstacleCollision();
    }
}

private void OnTriggerEnter2D(Collider2D other)
{
    if (IsObstacle(other.gameObject))
    {
        HandleObstacleCollision();
    }
}

// 死亡逻辑
public void Die()
{
    if (_isDead) return;
    _isDead = true;
    _respawnTimer = _respawnDelay;
    OnPlayerDied?.Invoke();
}

// 重生逻辑
private void Respawn()
{
    _isDead = false;
    transform.position = _lastSafePosition;
    if (_jumpController != null)
        _jumpController.ResetJumpState();
    var rb = GetComponent<Rigidbody2D>();
    if (rb != null)
        rb.velocity = Vector2.zero;
}
```

**配置参数:**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `_respawnDelay` | float | 1f | 死亡后重生延迟（秒） |
| `_enableDebugLog` | bool | false | 是否启用详细日志 |
| `_obstacleLayer` | LayerMask | 0 | 障碍物所在的 Layer（可选，LayerMask 为 0 时使用 Tag 检测） |

### 2. 障碍物碰撞检测

**方案:** 使用 Unity 2D 物理系统的碰撞检测

- 玩家：`BoxCollider2D` + `Rigidbody2D`
- 障碍物：`TilemapCollider2D` (已存在于 Obstacles Tilemap)
- 检测方式：
  - 优先：检查 `_obstacleLayer` LayerMask 配置
  - 后备：检查 GameObject Tag 是否为 "Obstacle"
